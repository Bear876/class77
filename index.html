To create a new project, you can createa new folder. If you want to code in html in that folder, you must write index.html. You can write style.css or function.js


We use the console so we can see the output of the code or it can help us debug it. You can print anything on the console by using console.log(). You can put whatever you want to print inside the round brackets.
